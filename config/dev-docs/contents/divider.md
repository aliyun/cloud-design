### Divider

| 参数          | 说明                                                    | 类型      | 默认值      |
| ----------- | ----------------------------------------------------- | ------- | -------- |
| dashed      | 是否为虚线                                                 | Boolean | false    |
| direction   | 线是水平还是垂直类型<br/><br/>**可选值**:<br/>'hoz', 'ver'            | Enum    | 'hoz'    |
| orientation | 分割线标题的位置<br/><br/>**可选值**:<br/>'left', 'right', 'center' | Enum    | 'center' |
